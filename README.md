# 버스 시간표 블로그 글 만들기

출발지, 도착지, 날짜를 입력하면 버스타고 운행 정보를 조회해서 네이버 블로그에 붙여넣기 좋은 시간표 글을 만들어 주는 페이지입니다.

## 실행 방법

```bash
npm start
```

실행 후 브라우저에서 `http://localhost:3000`을 열면 됩니다.

## 기능

- 버스타고 터미널 목록 검색
- 선택한 출발지 기준 도착지 검색
- 출발시간, 출발지, 도착지, 버스회사, 일반요금, 중고생요금, 아동요금, 예상소요시간 표 생성
- 블로그 글 텍스트 복사
- 블로그용 HTML 복사
- 네이버 접근 토큰이 있을 때 블로그 자동 게시 시도
- 네이버 글쓰기 창 열기
- 네이버 지도 기반 출발지-도착지 경로 이미지 생성 및 PNG 다운로드
- 인쇄용 화면 출력

## 주요 터미널 빠른 선택

출발지와 도착지 영역에는 주요 터미널 5개가 버튼으로 표시됩니다. 버튼을 클릭하면 해당 터미널이 바로 선택됩니다.

## 네이버 지도 경로 이미지

화면의 `네이버 지도 경로 그림` 영역에 Naver Cloud Platform의 Maps Static API 키를 입력한 뒤 `경로 그림 만들기`를 누르면 지도 이미지가 만들어집니다.

생성된 지도를 클릭하면 `출발지-도착지-naver-map.png` 파일로 다운로드됩니다.

환경 변수로 키를 넣고 실행할 수도 있습니다.

```bash
set NAVER_MAP_CLIENT_ID=발급받은_Client_ID
set NAVER_MAP_CLIENT_SECRET=발급받은_Client_Secret
npm start
```

현재 지도 이미지 기능은 빠른 선택 버튼에 등록된 주요 터미널 좌표를 기준으로 동작합니다. 직접 검색해서 선택한 터미널까지 지도 이미지로 만들려면 해당 터미널 좌표 데이터가 추가로 필요합니다.

## 네이버 블로그 자동 게시

화면의 `네이버 자동 게시` 영역에 네이버 개발자 접근 토큰을 입력하면 `https://openapi.naver.com/blog/writePost.json`으로 글 등록을 시도합니다.

토큰은 브라우저에만 저장됩니다. 서버 환경 변수로 올리려면 `NAVER_ACCESS_TOKEN` 값을 넣고 실행하면 됩니다.

```bash
set NAVER_ACCESS_TOKEN=발급받은_토큰
npm start
```

네이버 계정 또는 애플리케이션에 블로그 글쓰기 권한이 없거나 해당 API가 제한된 경우 자동 게시가 실패할 수 있습니다. 이때는 `네이버 글쓰기 열기` 버튼을 눌러 복사한 HTML을 글쓰기 화면에 붙여넣으면 됩니다.

## 참고

버스타고 API는 브라우저에서 직접 호출하면 CORS 제한이 생길 수 있어, 이 프로젝트의 `server.js`에서 서버가 대신 조회한 뒤 화면에 전달합니다.
