# 버스타고 블로그 시간표 만들기

출발지, 도착지, 날짜를 입력하면 버스타고 운행 정보를 조회해서 네이버 블로그에 붙여넣기 좋은 시간표 글을 만들어 주는 페이지입니다.

## 실행 방법

```bash
npm start
```

실행 후 브라우저에서 `http://localhost:3000`을 열면 됩니다.

## 기능

- 버스타고 터미널 목록 검색
- 선택한 출발지 기준 도착지 검색
- 출발시간, 출발지, 도착지, 버스회사, 일반요금, 중고생요금, 아동요금, 예상소요시간 표 생성
- 블로그 글 텍스트 복사
- 블로그용 HTML 복사
- 인쇄용 화면 출력

## 참고

버스타고 API는 브라우저에서 직접 호출하면 CORS 제한이 생길 수 있어, 이 프로젝트는 `server.js`에서 서버가 대신 조회한 뒤 화면에 전달합니다.
